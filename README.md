# منصة تعليمات العمل — السجل العقاري

## هيكل المشروع

```
├── index.html          ← المنصة (لا تعدّلها)
├── data/
│   ├── merge.md               ← دمج قطع عقارية
│   ├── split-plot.md          ← تجزئة القطع/المخططات
│   ├── split-units.md         ← فرز الوحدات العقارية
│   ├── re-split-units.md      ← إعادة فرز الوحدات
│   └── tasheer.md             ← التأشير في السجل العقاري
└── README.md
```

-----

## تحديث خدمة موجودة

1. افتح الملف على GitHub (مثلاً `data/merge.md`)
1. اضغط ✏️ Edit
1. عدّل ما تريد
1. اضغط **Commit changes**

-----

## إضافة خدمة جديدة

### الخطوة 1 — أنشئ الملف

في مجلد `data/` أنشئ ملفاً جديداً باسم الخدمة، مثل `correction.md`

### الخطوة 2 — انسخ هذا الهيكل

```markdown
---
id: correction
code: CR-WI-3.6
name: خدمة التصحيح
nameShort: التصحيح
icon: ✏️
color: #1d6fa4
status: active
version: 3.6
lastUpdate: DD/MM/YYYY
department: إدارة خدمات التسجيل
classification: عام
---

## الوصف
وصف الخدمة هنا.

## الكلمات المفتاحية
كلمة1، كلمة2، كلمة3

## الخطوات

### 1.0 | اسم الخطوة | المسؤول
- خطوة فرعية 1
- خطوة فرعية 2

### 2.0 | اسم الخطوة الثانية | المسؤول
- خطوة فرعية 1

## المستندات المطلوبة
- اسم المستند | إلزامي | ملاحظة اختيارية
- مستند آخر | اختياري

## متطلبات الموظف
- حساب أبشر | إلزامي
- صلاحية وصول | إلزامي

## أسباب الرفض
- R01 | سبب الرفض الأول
- R02 | سبب الرفض الثاني

## أسباب التعليق
- S01 | خلل تقني
- S02 | نقص مستندات

## ملاحظات مهمة
- warning | ملاحظة تحذيرية
- info | ملاحظة معلوماتية

## سجل الإصدارات
- 1.0 | DD/MM/YYYY | الإصدار الأولي | اسم المعتمد
```

### الخطوة 3 — أضف اسم الملف في index.html

افتح `index.html` وابحث عن:

```javascript
const FILES = [
  'data/merge.md',
  ...
```

أضف سطراً جديداً:

```javascript
  'data/correction.md',
```
